# PM Mock Interview Question Bank

This folder contains an extensive, app-ready repository of **Product Management mock interview questions + high-quality sample answers** across common interview categories:

- Product Design
- Estimation (Analytical)
- Behavioral
- Strategy
- Technical
- Execution

## How to use

- Pick a category in `categories/`
- Randomly select 1–3 prompts
- Timebox:
  - 2–3 minutes to clarify + structure
  - 10–15 minutes to answer
  - 3–5 minutes for follow-ups
- Grade yourself using the included rubric

## Files

- `schema.md`: field definitions and conventions
- `categories/`: one YAML file per category
